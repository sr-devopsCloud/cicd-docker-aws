## CI/CD Pipeline for Dockerized Application (AWS)

### Overview
This project demonstrates a simple CI/CD pipeline using GitHub Actions
to build and deploy a Dockerized application on AWS EC2.

### Tech Stack
- GitHub Actions
- Docker
- AWS EC2
- Linux

### Workflow
1. Code is pushed to GitHub
2. CI pipeline builds a Docker image
3. Application is deployed to an EC2 instance
4. App runs inside a Docker container

### Goal
Automate application build and deployment using DevOps best practices.
